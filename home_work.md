# Инструкция по работе с git

## Строки 
Чтобы выделить нумерованыне строки просто добавьте перед ними номер строки. Например:
1. Первая строка
2. Вторая строка
3. Третья строка
Чтобы выделить ненумерованные строки добавьте символ (*) перед началом строки. Например:
* Первая строка 
* Вторая строка
* Третья строка
